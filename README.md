## Начало работы 
Скоприовать файл ./src/config.default.js в ./src/config.js

Изменить HOST на нужный, пример ```HOST: "http://event-radar.ru/api"```:  

## Скрипты 

### npm run dev
Запуск сервера разработчика

### npm run build
Сборка проекта для прода

### npm run deploy
Запуск сервера на проде
